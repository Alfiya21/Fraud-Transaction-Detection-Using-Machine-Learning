About the project

What is TxnGuard?

An end-to-end machine learning pipeline built to detect fraudulent financial transactions with high precision and full explainability.

Financial fraud causes billions in losses annually. TxnGuard tackles this by training ensemble models on over 1.75 million transactions, including real-world fraud scenarios: high-value fraud, terminal compromise, and customer credential leakage.

The system engineers time-based, customer-behavioral, and terminal-risk features, then uses SMOTE to address severe class imbalance. XGBoost and Random Forest classifiers are trained and benchmarked — XGBoost achieves a ROC-AUC of 0.9749.

Every prediction is explained using SHAP (SHapley Additive exPlanations), making the model transparent and trustworthy for real-world fraud flagging decisions.

What the system does

From raw transactions to explainable fraud signals — TxnGuard covers the full detection pipeline.

Real-time prediction

Submit a transaction and receive an instant fraud or legitimate verdict with a confidence probability score.

SHAP explainability

Every prediction is accompanied by SHAP values showing exactly which features drove the model's decision.

Behavioral analysis

Customer spending patterns over 7 and 30-day windows are analyzed to detect anomalous transaction behavior.

Terminal risk scoring

Terminal-level fraud history and average transaction amounts are used as risk signals in the feature space.

Time-aware features

Transaction hour, day of week, and weekend flags capture temporal fraud patterns invisible to simple models.

Model benchmarking

XGBoost and Random Forest are evaluated side-by-side. XGBoost wins on recall and ROC-AUC for imbalanced data.
